## Cài Đặt

### Yêu Cầu

- Node.js (phiên bản 14 trở lên)
- npm (phiên bản 6 trở lên)

### Bước Cài Đặt

1. Clone repository:

    ```bash
    git clone https://github.com/your-username/your-project.git
    ```

2. Di chuyển vào thư mục dự án:

    ```bash
    cd your-project
    ```

3. Cài đặt các gói phụ thuộc:

    ```bash
    npm install
    ```
